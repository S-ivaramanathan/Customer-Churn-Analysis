# Customer Churn Analysis

## Project Overview
This project analyzes customer churn in a telecom company using Python, SQL, Machine Learning, and Power BI. The objective is to identify customers who are likely to leave the company and understand the factors influencing customer churn.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- SQL (MySQL)
- Power BI

## Machine Learning Models
- Logistic Regression
- Decision Tree
- Random Forest
- K-Nearest Neighbors (KNN)
- Naive Bayes
- Support Vector Machine (SVM)

## Project Workflow
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Feature Engineering
- Model Training
- Model Evaluation
- SQL Analysis
- Power BI Dashboard

## Dataset
Telco Customer Churn Dataset
